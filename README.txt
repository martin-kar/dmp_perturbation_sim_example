Matlab simulation code written by Martin Karlsson and Fredrik Bagge Carlson, 
Dept. Automatic Control, Lund University, Lund, Sweden.
martin.karlsson@control.lth.se

Code example for the scientific paper "Two-Degree-of-Freedom Control for 
Trajectory Tracking and Perturbation Recovery during Execution of Dynamical 
Movement Primitives" (IFAC 2017) by Martin Karlsson, Fredrik Bagge Carlson, 
Anders Robertsson and Rolf Johansson. The main.m file produces two plots: 
one with a stopping perturbation, and one with a moving perturbation.
